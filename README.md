## How to use:
  1) Clone this repository.

  2) `cd` to the cloned folder.

  3) Run  ```npm install``` in the terminal.

  4) Start the Development server by typing `npm start` in the terminal.
 
  5) Go to [http://localhost:8080](http://localhost:8080).

  6) Register.

  7) Have fun =)).
